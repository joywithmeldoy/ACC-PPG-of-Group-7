# ACC-PPG-of-Group-7
Imperial college London Lab in Sensor System Group 7 ACC+PPG sensor
This repository contains two sensor modules used in our wearable sensing system:

ACC module (accelerometer board)

PPG module (photoplethysmography board)

It includes schematics, PCB outputs, Gerbers, BOMs, firmware examples, and the corresponding component datasheets for documentation and review.
